

1- ¿Qué contiene el archivo AndroidManifest.xml?

Contiene la configuración principal de la aplicación Android, como:

Información general de la app.
Las actividades (pantallas) de la aplicación.
Permisos.
Icono y tema de la app.
Configuración de inicio.

En mi caso, define una aplicación llamada HolaAndroid y registra la MainActivity como pantalla principal.

-----------------------------------------------------------------------------------------------------------------

2- ¿Qué es el atributo android:name en <activity>?


El atributo android:name indica la clase Activity que Android debe ejecutar.
